# Password Checker

A beginner-friendly Python project that checks password strength and detects if a password has been found in a breach list. This project helps users understand what makes a strong password and encourages safer password practices.

---

## Features

- **Password Strength Checks:**
  - Minimum 8 characters
  - At least one uppercase letter
  - At least one lowercase letter
  - At least one number
  - At least one special character (`@$!%*?&#`)
  - No consecutive repeated characters
- **Breach Detection:**
  - Checks passwords against a local `passwords.txt` breach list
- **Interactive Feedback:**
  - Prints messages guiding the user to improve their password
- **Strength Levels:**
  - Weak, Medium, or Strong based on scoring rules
